Catalogmetadata

A website collecting the metadata (API calls or file downloads) offered by data catalogs in Germany

The following links should allow you to download the metadata of the data catalog for each city/state. Please note that in some cases the download is only an overview and that further info may be available with further API calls or visits to the links contained in the meta data. If your city is missing, please add it!
